# Grand Strategy Chess
Welcome to a world of chess, just a little a more complicated.

## Get Developing
1. Download the tools listed in the tools section (use the given version for stable development)
2. Go to ```cd ./game``` in the terminal
3. Perform ```cargo build``` in the terminal

You are now setup to start developing the GSC.

## Get Documenting
1. For progress on features and fixes, update the issues board with sub issues and closing issues through PRs.
2. For design and architecture changes to the game, use Obsidian to review the notes.

## Tools
* Language - Rust - 1.911
* Engine - Bevy - 0.18.1
* IDE - Visul Studio Code - latest
